# Didik App - Aplikasi Review Magang (ETS Mobile Programming)

Aplikasi **Didik** adalah platform review tempat magang yang dibangun untuk memenuhi tugas **Evaluasi Tengah Semester (ETS)** mata kuliah Pemrograman Perangkat Bergerak. Aplikasi ini memungkinkan pengguna untuk melihat, menambah, dan membaca detail pengalaman magang mahasiswa secara terstruktur.

## 🚀 Daftar Fitur
1. **Splash & Welcome Screen:** Navigasi awal aplikasi.
2. **Register:** Pendaftaran akun baru.
3. **Login:** Autentikasi menggunakan email dan password.
4. **Dashboard:** Menampilkan list review magang menggunakan RecyclerView.
5. **Add Review:** Form untuk menambah data review baru.
6. **Detail Review:** Menampilkan detail lengkap dari review yang dipilih.
7. **Profile:** Informasi pengguna yang sedang aktif.
